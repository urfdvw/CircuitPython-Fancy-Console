# Components

Serial input
- used for send data to cpy
- generic input (plain text)
- no newline supported, single line only
- shortcut
    - send

Repl input
- used for send python code to cpy
- python highlighting
- with history
- shortcuts
    - python editor shortcut
    - send shortcut
        - swapable with newline
    - ctrl-c/ctrl-d
    - history retract

Title bar
- show the most recent title
